Qu’est-ce que l’HTML ?

HTML (HyperText Markup Langage) est le bloc de construction le plus basic du WEB. Il est la plupart du temps accompagner par d’autre langage comme le CSS pour l’apparence/la présentation, ou le comportement par le JavaScript (JS). Il a débuté en 1991.


Exemple de balisage : 

                <tagname> Contenue de la balise. </tagname>

Un document HTML simple: 

                    <!DOCKTYPE html> (1)
                    <html> (2)
                    <head> (3)
                    <title> Page Title </title> (4)
                    </head> (3)
                    
                    <body> (5)
                        <h1> My first Heading </h1> (6)
                        <p> My first Paragraph </p> (7)
                    </body> (5)
                    </html> (2)

Exemple expliqué:

(1): La déclaration qui définit que ce document est un document HTML 5.
(2): L’élément racine d’une page HTML.
(3): Balise contenant de meta-informations sur la page HTML.
(4): L’élément qui spécifie un titre pour la page, il sera affiche sur la barre de titre de navigation.
(5): Définit le corps du document et est un contenue pour tous les objets visibles tels que les titres, les paragraphes et les images.
(6): Définit un grand titre.
(7): Définit un paragraphe.
